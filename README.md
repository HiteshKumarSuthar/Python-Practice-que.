# Python Practice Questions 🚀

Welcome to this repository! This collection contains practice questions designed to strengthen your understanding of **Python programming fundamentals**. These exercises are perfect for beginners who want to improve their problem-solving skills through consistent practice.

## 📚 Topics Covered

* 🔹 If-Else Conditional Statements
* 🔹 Loops (`for` & `while`)
* 🔹 Strings

## 🎯 Purpose

The main goal of this repository is to:

* Build a strong foundation in Python.
* Improve logical thinking and problem-solving skills.
* Practice coding with beginner-friendly questions.
* Prepare for coding interviews and programming contests.

## 📌 Who Is This For?

This repository is suitable for:

* Beginners learning Python.
* Students practicing programming concepts.
* Anyone looking to strengthen their coding fundamentals.

## 💡 How to Use

1. Pick a question.
2. Try solving it on your own.
3. Compare your solution with the provided one (if available).
4. Repeat consistently to improve your coding skills.
